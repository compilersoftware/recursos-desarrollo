### Gearsystem

https://github.com/drhelius/Gearsystem

Gearsystem is a cross-platform Sega Master System / Game Gear / SG-1000 emulator written in C++ that runs on Windows, macOS, Linux, iOS, Raspberry Pi and RetroArch.
